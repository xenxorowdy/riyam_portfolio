## Description

A Ticket booking app where you can validate your journey.
Buy Passes and tickets and made your journey smooth.

## Features

- Get ticket history
- Buy Passes

## Tech Stack

- React Native
- MongoDB

## Getting Started

1. Clone repository

```bash
git clone https://github.com/xenxorowdy/tummoc
cd tummoc
```

2. Install dependencies

```bash
npm install
# or
yarn install
```

3. Start development server

```bash
npm run start
# or
yarn start
```
